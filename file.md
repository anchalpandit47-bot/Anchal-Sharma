# Dr. Anchal Sharma - Academic Website

This is a personal academic website for Dr. Anchal Sharma, Assistant Professor of Psychology.

## Files
- `index.html` - Main website content
- `style.css` - Website styling
- `README.md` - Project information

## How to use
1. Download all files.
2. Put them in one folder.
3. Open `index.html` in your browser.

## Upload to GitHub Pages
1. Create a new repository on GitHub.
2. Upload `index.html` and `style.css`.
3. Go to repository **Settings** > **Pages**.
4. Under **Source**, select the main branch.
5. Save.
6. Your website will be live on GitHub Pages.
